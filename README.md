# BallRunner AI Suite

This repository bundles:

- `game/`: Unity gameplay project (`BallRunnerHybrid`)
- `orchestrator/`: multi-model task orchestrator and automation scripts

## Included
- Gameplay systems (core loop, gates, obstacles, economy, ads hooks)
- Runtime level builder and bootstrap flow
- Orchestrator with model routing, dispatch, health checks, and state files

## Notes
- Temporary artifacts and runtime logs are excluded.
